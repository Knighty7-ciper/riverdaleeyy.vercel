================================================================================
                              CUSTOMER WORKFLOW
================================================================================

1. Customer visits website
2. Browses destinations (grouped by countries)
3. Views individual package details
4. Submits inquiry with personal details
5. Receives verification ID (RVD-YYYYMMDD-XXXX)
6. Admin receives email notification
7. Admin calls customer for verification
8. Admin generates quotation
9. Manual payment tracking until Pesapal integration
